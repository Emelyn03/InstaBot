# InstaBot

Bot de automatización para Instagram desarrollado con Python y Flask.

## Descripción

InstaBot es una aplicación web que permite interactuar automáticamente con publicaciones de Instagram mediante:

- Likes automáticos
- Comentarios automáticos
- Inicio de sesión mediante instagrapi
- Interfaz web simple y funcional

El proyecto fue creado con fines educativos y de aprendizaje sobre automatización, desarrollo backend y APIs.

## Instalación

### 1. Clonar repositorio

```bash
git clone https://github.com/Emelyn03/InstaBot.git
```

### 2. Entrar al proyecto

```bash
cd InstaBot
```

### 3. Crear entorno virtual

```bash
python -m venv venv
```

### 4. Activar entorno virtual

#### Windows

```bash
.\venv\Scripts\activate
```

---

## Instalar dependencias

```bash
pip install flask instagrapi
```

---

## Ejecutar aplicación

```bash
python app.py
```
